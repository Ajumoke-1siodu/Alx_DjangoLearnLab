## Overview 
LibraryProject is a Django-based application designed to help you gain familiarity with Django by setting up a development environment, creating a project, and building a simple app (`bookshelf`) with a `Book` model.
 This project demonstrates basic CRUD operations using Djangoâ€™s ORM. 
## Features 
- Django project setup (`LibraryProject`) 
- Custom app: `bookshelf` 
- `Book` model with fields: - `title` (CharField, max_length=200) 
- `author` (CharField, max_length=100) 
- `publication_year` (IntegerField) 
- CRUD operations demonstrated in the Django shell: - Create - Retrieve - Update - Delete
